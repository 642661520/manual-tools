飞书机器人

平台使用飞书机器人应用发送消息通知。

配置要求

需要在飞书开放平台创建企业自建应用，并在平台环境变量中配置：

  • FEISHU_APP_ID — 应用的 App ID
  • FEISHU_APP_SECRET — 应用的 App Secret
  • FEISHU_HOST — 飞书 API 主机（默认 https://open.feishu.cn）

权限要求

飞书应用需要以下权限：

  • 获取用户信息（用于 OAuth 登录）
  • 发送消息（用于通知推送）
  • 获取用户 ID（用于 Open ID 查询）
