版本对比界面

可以对比同一目录的任意两个已发布版本，查看文档变更。

操作方式

  1. 在目录详情页面进入版本管理
  2. 选择两个要对比的版本（左侧为旧版本，右侧为新版本）
  3. 点击对比按钮

对比视图

对比界面分为两部分：

  • 左侧导航 — 显示有变更的章节列表（新增/修改/删除标记）
  • 右侧内容 — 并排或统一显示两个版本的差异，修改部分高亮标记
图片占位：版本对比界面：左侧变更列表和右侧差异内容视图

差异查看

版本对比使用文本差异算法（diff）精确展示两个版本之间的内容变化。

差异类型

  • 新增 — 新版本中新增的章节或段落（绿色高亮）
  • 删除 — 新版本中删除的章节或段落（红色高亮）
  • 修改 — 内容发生变化的章节（黄色高亮）
  • 未变更 — 两版本一致的章节（无标记）

筛选功能

可以按差异类型筛选，只看某种类型的变更。这对于审查大量变更时非常有用。
