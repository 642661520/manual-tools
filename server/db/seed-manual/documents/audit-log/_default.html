日志查看

在系统设置 → 操作日志中可以查看系统中的所有重要操作记录。

日志内容

每条日志记录包含：

  • 操作人和用户名
  • 操作类型（创建、更新、删除等）
  • 操作目标类型和 ID
  • 操作详情
  • 操作时间

筛选功能

支持按操作类型筛选日志，如只查看删除操作或权限变更操作。日志默认按时间倒序排列，最新的操作显示在前面。

操作记录

系统自动记录以下类型的操作：

记录范围

  • 用户管理 — 创建用户、删除用户、修改角色
  • 项目管理 — 创建项目、删除项目、修改设置
  • 成员管理 — 添加成员、移除成员、修改角色
  • 功能管理 — 创建/编辑/删除功能、批量导入
  • 文档操作 — 状态变更、提交审核、审核通过/驳回
  • 目录管理 — 创建/编辑/删除目录、版本发布
  • 数据操作 — 项目导入导出

日志保留

审计日志目前存储在数据库中，无自动过期机制。建议定期检查日志量，必要时手动清理旧日志。
