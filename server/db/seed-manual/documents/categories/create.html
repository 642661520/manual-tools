创建分类

分类（Category）用于对功能进行分组管理，在功能列表中以不同颜色标识区分。

操作步骤

  1. 进入功能列表页面
  2. 点击页面右上角的管理分类按钮
  3. 在分类管理面板顶部的输入框中填写分类名称，选择颜色，点击添加
  4. 填写分类信息：
    • 分类名称 — 必填，如"项目管理"、"文档编辑"
    • 颜色标识 — 选填，默认为靛蓝色（#6366f1）
  5. 点击确认

颜色选择

使用浏览器原生颜色选择器自由选取任意颜色。建议为不同分类使用不同颜色以方便视觉区分。颜色会在功能列表的分类标题左侧显示为圆点标识。
