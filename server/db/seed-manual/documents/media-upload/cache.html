远程图片缓存

从外部网页粘贴内容时，如果包含远程图片链接，平台会自动将图片下载并缓存到本地。

缓存机制

  1. 粘贴包含远程图片链接的 HTML 内容
  2. 系统识别 <img src="https://..."> 标签
  3. 自动下载远程图片到本地缓存目录
  4. 替换图片链接为本地路径

缓存管理

  • 远程缓存有 TTL 过期时间（默认 7 天，可通过 REMOTE_CACHE_TTL_DAYS 调整）
  • 缓存文件最大 50MB（可通过 REMOTE_CACHE_MAX_FILE_MB 调整）
  • 系统启动时会自动清理过期缓存
