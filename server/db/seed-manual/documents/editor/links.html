链接与引用

编辑器支持多种类型的链接。

超链接

选中文本后点击工具栏链接按钮，输入 URL 即可创建外部超链接。点击已有链接可以修改 URL 或取消链接。

交叉引用

交叉引用（Cross Reference）用于在同一手册内引用其他功能或章节，构建文档内部导航。点击工具栏交叉引用按钮：

  1. 在弹出的选择器中搜索目标功能
  2. 选择要引用的功能或具体章节
  3. 引用的显示文本自动使用功能/章节标题

交叉引用在预览和导出时会自动渲染为可点击的内部链接。

链接样式

所有链接默认显示为蓝色下划线样式，鼠标悬停时显示链接目标预览。
