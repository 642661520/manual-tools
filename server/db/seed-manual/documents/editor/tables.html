表格

编辑器内置完整的表格功能，支持创建和编辑结构化表格。

插入表格

点击工具栏中的表格按钮，使用网格选取器选择行列数（最大 10×10），点击即可插入。

表格操作

选中表格或单元格后，会出现浮动工具栏，支持以下操作：

  • 在上方/下方插入行
  • 在左侧/右侧插入列
  • 删除选中行
  • 删除选中列
  • 合并/拆分单元格
  • 切换表头行/列（表头单元格有特殊样式）
  • 删除整个表格

注意事项

  • 表头行会在导出的 PDF 和 Markdown 中保留标识
  • 表格宽度自适应编辑区域
  • 单元格内支持基本的内联格式（加粗、斜体等）
