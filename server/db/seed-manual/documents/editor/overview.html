编辑器界面

Manual Tools 的编辑器基于 TipTap（ProseMirror 的 Vue 3 封装），提供所见即所得（WYSIWYG）的编辑体验。

界面布局

编辑器页面分为三个区域：

  1. 左侧导航树 — 显示当前功能的所有章节，点击切换编辑目标
  2. 中间编辑区 — TipTap 富文本编辑器主体
  3. 底部状态栏 — 显示文档状态、字数统计、审核信息

工具栏

编辑器顶部工具栏提供完整的文本格式化功能，按功能分组排列：

  • 文本样式 — 段落、标题 1-6
  • 内联格式 — 加粗、斜体、下划线、删除线、文字颜色、高亮
  • 列表 — 无序列表、有序列表、任务列表
  • 插入 — 图片、视频、表格、链接、交叉引用
  • 工具 — 查找替换、撤销重做

快捷操作

  • 支持 Markdown 语法快捷输入（如 # 开头自动转为标题）
  • 支持粘贴 Markdown 内容自动转换
  • 支持拖拽图片到编辑区直接上传
