项目概念

在 Manual Tools 平台中，项目（Project）是数据组织和隔离的基本单位。每个项目拥有独立的：

  • 分类系统 — 自定义的功能分类和颜色标记
  • 功能集合 — 文档编写的功能骨架和章节结构
  • 文档内容 — 基于 Y.js 协同编辑的富文本文档
  • 目录编排 — 将功能组装为完整手册的目录
  • 成员权限 — 独立的项目成员和角色管理

平台首次启动时会自动创建一个默认项目，包含本使用手册的完整内容，方便您快速上手。

项目的作用

通过多项目机制，您可以：

  1. 为不同的产品线或业务模块创建独立项目
  2. 在同一平台中管理多个文档工程
  3. 按项目粒度控制用户的访问和操作权限
  4. 独立导出和迁移单个项目的数据
