项目角色详解

项目角色控制用户在特定项目内的操作权限。

PM（项目负责人）

  • 管理项目设置和审核链
  • 管理项目成员
  • 创建和管理分类、功能、目录
  • 预览和发布手册版本
  • 直接通过文档审核
  • 项目数据导入导出

Writer（编辑者）

  • 查看项目中的所有功能和分类
  • 编辑文档内容（富文本编辑器）
  • 参与协同编辑
  • 提交文档审核
  • 查看手册预览

Viewer（查阅者）

  • 查看功能列表和分类
  • 查看文档内容（只读模式）
  • 查看手册预览和已发布版本
