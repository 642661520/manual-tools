系统角色详解

系统角色控制用户在平台级别的操作权限。

Admin 权限

  • 创建和删除项目
  • 管理所有项目的成员
  • 创建和管理用户
  • 修改其他用户的系统角色
  • 系统级数据导入导出和备份
  • 查看审计日志
  • 对所有项目自动拥有 PM 权限

Member 权限

  • 被添加到项目后按项目角色工作
  • 修改自己的个人资料和密码
  • 绑定飞书账号
  • 查看自己的待办任务

Guest 权限

  • 登录并查看公开文档
  • 修改自己的个人资料和密码
  • 无法被添加到任何项目
