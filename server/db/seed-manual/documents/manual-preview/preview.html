预览模式

在目录编辑页面，可以实时预览最终手册的效果。

操作方式

点击目录编辑页面顶部的预览按钮，系统会：

  1. 按照目录编排顺序读取所有功能章节的文档内容
  2. 将内容组装为完整的 Markdown/HTML 手册
  3. 在预览页面中渲染展示

预览功能

  • 目录导航 — 左侧自动生成带锚点的章节目录
  • 按状态筛选 — 可以只看已审核通过的内容
  • 按需刷新 — 文档内容变化后刷新预览即可看到最新效果

导出与下载

预览页面工具栏提供多种导出选项：

  • 下载 MD — 将当前版本导出为 Markdown ZIP 包（含图片等资源）
  • 下载 PDF — 将当前版本导出为 PDF 文件
  • 下载离线站点 — 导出当前目录下所有公开版本的静态站点 ZIP 包，解压后可直接用 nginx 部署到内网环境

其中 MD 和 PDF 导出基于当前所选版本（或实时内容），而离线站点是 catalog 级别的操作，始终导出所有已发布的公开版本。
