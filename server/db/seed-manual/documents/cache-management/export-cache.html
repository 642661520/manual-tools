导出缓存管理

每次导出 PDF 或 Markdown 文件时，系统会缓存导出结果以便后续快速下载。

缓存内容

导出缓存包含：

  • PDF 格式的手册导出文件
  • Markdown 格式的 ZIP 导出包
  • 与每个目录（Catalog）关联，同一目录重新导出时复用缓存

查看缓存

在存储管理页面的导出缓存区域可以看到：

  • 缓存文件总数和总大小
  • 每个缓存条目对应的目录名称、导出类型和创建时间
  • 支持直接下载缓存文件

手动失效

如果目录内容已更新但缓存未自动刷新，可以点击清除此目录缓存使特定目录的导出缓存失效，下次导出时将重新生成。

配置

通过环境变量调整缓存策略：

  • EXPORT_CACHE_TTL_DAYS — 导出缓存过期天数（默认 30）
  • CACHE_DIR — 缓存根目录路径
