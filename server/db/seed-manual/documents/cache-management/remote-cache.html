远程资源缓存

在编辑器中粘贴外部图片链接时，平台会自动将图片下载并缓存到本地，避免外部链接失效导致图片无法显示。

工作原理

  1. 在编辑器中粘贴包含外部图片的 URL 或内容
  2. 系统自动下载图片并计算 SHA-256 哈希值
  3. 图片存储在 data/cache/remote/{hash前2位}/{完整hash}.{ext}
  4. 文档中的图片链接替换为本地 /uploads/images/... 路径

查看缓存

在数据管理页面的远程缓存区域可以看到：

  • 已缓存的远程图片数量
  • 每条缓存记录的原始 URL、文件类型、大小和缓存时间
  • 支持预览缓存图片

管理操作

  • 删除单条 — 删除特定远程资源的缓存
  • 清除全部 — 清除所有远程缓存（不影响已转换为本地路径的文档内容）

配置

  • REMOTE_CACHE_TTL_DAYS — 远程缓存过期天数（默认 7）
  • REMOTE_CACHE_MAX_FILE_MB — 单文件大小上限（默认 50MB）
