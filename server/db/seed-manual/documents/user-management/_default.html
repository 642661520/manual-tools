创建用户

系统管理员（Admin）可以在设置页面创建新用户。

操作步骤

  1. 进入系统设置 → 用户管理标签页
  2. 点击新建用户按钮
  3. 填写用户信息：
    • 用户名 — 必填，用于登录
    • 显示名称 — 可选，在界面中展示
    • 密码 — 必填，初始登录密码
    • 角色 — 选择 Admin / Member / Guest
  4. 点击创建

密码要求

初始密码建议使用强密码。用户登录后可以在个人资料页面自行修改密码。

图片占位：user-management - create-user

查看用户

在用户管理页面可以查看系统中所有已注册用户。

用户列表

列表显示每个用户的：

  • 用户名和显示名称
  • 系统角色（Admin / Member / Guest）
  • 飞书绑定状态（是否已绑定飞书账号）
  • 创建时间

筛选

支持在前端按用户名快速筛选，定位目标用户。

图片占位：user-management - list

删除用户

系统管理员可以删除不再需要的用户账号。删除操作不可逆，请谨慎操作。

注意事项

  • 删除用户前，建议先从所有项目中移除该用户
  • 该用户创建的文档内容不会丢失（文档归属保留）
  • 该用户的审核任务将被取消
  • 删除后该用户无法再登录系统
  • 内置管理员账号受保护 — 系统初始管理员不可删除

操作步骤

  1. 进入系统设置 → 用户管理
  2. 在用户列表中找到目标用户
  3. 点击删除按钮
  4. 在确认对话框中确认删除

限制

  • 仅 Admin 角色可以删除用户
  • 不能删除自己的账号
图片占位：删除用户确认对话框：显示将被删除的用户名

系统角色说明

每个用户有一个系统级角色，控制其在平台中的整体权限。

角色定义

角色 权限范围
Admin（系统管理员） 拥有所有权限：管理项目、用户、系统设置、数据导入导出。对所有项目自动拥有 PM 权限
Member（普通用户） 可以加入项目并按项目角色工作。无法访问系统设置
Guest（访客） 只能查看已登录用户可访问的公开文档。无法加入项目

角色修改

Admin 可以修改其他用户的系统角色。所有用户都无法修改自己的角色。

图片占位：user-management - roles
