删除用户

系统管理员可以删除不再需要的用户账号。删除操作不可逆，请谨慎操作。

注意事项

  • 删除用户前，建议先从所有项目中移除该用户
  • 该用户创建的文档内容不会丢失（文档归属保留）
  • 该用户的审核任务将被取消
  • 删除后该用户无法再登录系统
  • 内置管理员账号受保护 — 系统初始管理员不可删除

操作步骤

  1. 进入系统设置 → 用户管理
  2. 在用户列表中找到目标用户
  3. 点击删除按钮
  4. 在确认对话框中确认删除

限制

  • 仅 Admin 角色可以删除用户
  • 不能删除自己的账号
