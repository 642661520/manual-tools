飞书绑定

绑定飞书账号后，可以使用飞书登录并接收消息通知。

绑定步骤

  1. 进入个人资料页面
  2. 在飞书绑定区域点击绑定飞书账号
  3. 跳转到飞书 OAuth 授权页面
  4. 在飞书页面确认授权
  5. 自动返回平台，绑定完成

绑定后

  • 登录页面增加"飞书登录"选项
  • 审核通知等消息推送到飞书
  • 可以随时在个人资料页面解除绑定

前提条件

系统需要配置飞书应用（FEISHU_APP_ID、FEISHU_APP_SECRET、FEISHU_REDIRECT_URI 环境变量）才能使用飞书绑定功能。
