修改显示名称

显示名称是您在平台中的展示名称，会出现在协同编辑的光标提示和审核日志中。

修改步骤

  1. 点击右上角用户头像 → 个人资料
  2. 在显示名称字段中输入新名称
  3. 点击保存

注意事项

  • 显示名称不等于用户名，不影响登录
  • 建议使用真实姓名或常用的称呼
  • 修改后所有协作者会立即看到新名称
图片占位：profile - display-name

修改用户名

用户名是登录系统时使用的账号标识。

修改步骤

  1. 进入个人资料页面
  2. 在用户名字段中输入新用户名
  3. 点击保存

注意事项

  • 用户名必须是系统中唯一的，不可与其他用户重复
  • 修改用户名后，下次登录需使用新用户名
  • 修改用户名会使所有旧 JWT Token 失效，需要重新登录
图片占位：profile - username

修改密码

定期修改密码是保护账号安全的好习惯。

修改步骤

  1. 进入个人资料页面
  2. 在修改密码区域填写当前密码和新密码
  3. 点击修改密码

安全建议

  • 密码长度不少于 8 位
  • 建议包含字母、数字和特殊字符
  • 不要使用与其他平台相同的密码
  • 修改密码后所有旧 JWT Token 失效，需要重新登录
图片占位：profile - password

飞书绑定

绑定飞书账号后，可以使用飞书登录并接收消息通知。

绑定步骤

  1. 进入个人资料页面
  2. 在飞书绑定区域点击绑定飞书账号
  3. 跳转到飞书 OAuth 授权页面
  4. 在飞书页面确认授权
  5. 自动返回平台，绑定完成

绑定后

  • 登录页面增加"飞书登录"选项
  • 审核通知等消息推送到飞书
  • 可以随时在个人资料页面解除绑定

前提条件

系统需要配置飞书应用（FEISHU_APP_ID、FEISHU_APP_SECRET 环境变量）才能使用飞书绑定功能。

图片占位：profile - feishu-bind
