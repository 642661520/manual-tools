项目导出

支持将整个项目的数据导出为 ZIP 文件，用于数据迁移和备份。

导出内容

导出 ZIP 包含：

  • 项目基本信息（名称、描述、审核链配置）
  • 所有分类和功能定义
  • 所有文档内容（Y.js 增量数据和快照）
  • 所有目录和发布版本
  • 文档中引用的上传文件（图片、视频）

操作步骤

  1. 进入项目设置 → 数据管理
  2. 点击导出项目
  3. 系统估算导出大小并显示预览
  4. 确认后开始导出（异步任务）
  5. 导出完成后下载 ZIP 文件

任务管理

导出是异步任务，支持进度查询和历史记录查看。导出任务 24 小时后自动过期清理。
