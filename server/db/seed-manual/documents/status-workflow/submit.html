提交审核

文档编写完成后，您可以将其提交到审核流程。

操作步骤

  1. 在编辑器中完成文档编写
  2. 点击底部状态栏的提交审核按钮
  3. 在弹出的对话框中：
    • 填写提交说明（可选）
    • 确认指派给审核人（如已配置审核链）
  4. 点击确认提交

提交后

  • 文档状态变为 pending_review
  • 审核人会收到待办通知（如已配置飞书集成）
  • 文档变为只读状态，直到审核完成
  • 提交记录会写入审核日志

注意事项

  • 请确保文档内容完整、格式规范后再提交
  • 提交后无法修改内容（除非被驳回或主动退回）
