Markdown 导出

支持将手册导出为 Markdown 文件，方便导入到其他文档系统或进行版本控制。

导出方式

在版本详情或预览页面，点击导出 Markdown。系统会：

  1. 读取所有文档内容
  2. 将 HTML 内容转换为 Markdown 格式
  3. 生成目录层次结构
  4. 打包为 ZIP 文件供下载

输出内容

ZIP 文件包含按目录结构组织的 Markdown 文件，每个功能一个文件夹，每个章节一个 .md 文件。

图片占位：markdown-export - export-md

图片引用处理

导出 Markdown 时，文档中的图片引用会自动处理。

处理方式

  • Markdown 中使用相对路径引用图片
  • 图片文件一同打包到 ZIP 中
  • 导出后的 Markdown 可以在本地 Markdown 编辑器中正常查看

注意事项

  • 视频文件会一同打包到 ZIP 中
  • 交叉引用会转换为带章节锚点的可点击链接
