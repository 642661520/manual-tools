添加成员

项目负责人（PM）可以在项目设置中添加新成员。

操作步骤

  1. 进入项目设置 → 成员管理标签页
  2. 点击添加成员按钮
  3. 从用户下拉列表中选择目标用户
  4. 从下拉列表中选择目标用户
  5. 选择成员角色（PM / Writer / Viewer）
  6. 点击确认添加

前提条件

  • 目标用户必须已在系统中注册
  • 如果目标用户已是项目成员，则更新其角色
🖼️
图片占位：project-members - add

移除成员

项目负责人可以从项目中移除成员。

操作步骤

  1. 在成员列表中找到要移除的成员
  2. 点击该成员行右侧的移除按钮
  3. 在确认对话框中确认移除操作

注意事项

  • 移除成员后，该用户将无法访问该项目的数据
  • 该用户已创建的文档内容不会丢失
  • 该用户在该项目中的审核任务将被取消
  • 如果该用户是审核链中的审核人，建议先调整审核链配置
🖼️
图片占位：project-members - remove

项目角色说明

每个项目支持三种成员角色，权限从高到低排列：

PM（项目负责人）

  • 管理项目成员（添加、移除、修改角色）
  • 配置审核链
  • 创建和管理分类
  • 创建和管理功能
  • 创建和管理目录
  • 预览和发布手册版本
  • 导入和导出项目数据

Writer（编辑者）

  • 查看项目中的功能和分类
  • 编辑文档内容（富文本编辑器）
  • 参与协同编辑
  • 提交文档审核
  • 查看手册预览

Viewer（查阅者）

  • 查看项目中的功能和分类
  • 查看文档内容（只读）
  • 查看手册预览
  • 查看已发布的静态文档站点

系统管理员（Admin）对所有项目拥有完整权限，不受项目角色限制。
