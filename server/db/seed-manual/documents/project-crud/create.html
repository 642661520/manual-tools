创建项目

系统管理员（Admin 角色）可以在设置页面创建新项目。

操作步骤

  1. 点击顶部导航栏右侧用户菜单中的系统设置（盾牌图标，仅 Admin 可见）
  2. 进入项目管理标签页
  3. 点击新建项目按钮
  4. 在弹出的对话框中填写：
    • 项目名称 — 必填，用于标识项目
    • 项目描述 — 可选，简要说明项目用途
  5. 点击确认创建

自动配置

项目创建成功后，系统会自动：

  • 生成唯一的项目 ID（8 位随机字符串）
  • 将创建者添加为项目的 PM（项目负责人）
  • 创建项目级别的空数据空间

新项目创建后为空项目，您可以立即开始添加分类和功能。
