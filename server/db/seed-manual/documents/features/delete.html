删除功能

谨慎使用此操作。删除功能会级联删除该功能下所有章节的文档内容。

操作步骤

  1. 在功能列表中，点击功能行右侧的删除按钮
  2. 仔细阅读删除确认对话框中的警告信息
  3. 确认删除

级联影响

  • 该功能下的所有文档会被删除（包括所有 Y.js 编辑历史）
  • 文档快照和增量更新记录会被清除
  • 如果该功能被包含在某个目录中，目录中会残留对该功能的无效引用，需要手动从目录中移除

建议

在删除功能前，请先：

  1. 检查该功能是否被目录引用，如有则先从目录中移除
  2. 导出项目数据作为备份
  3. 确认团队成员不再需要该功能的文档
  4. 删除后建议重建搜索索引（通过重启服务器或调用 API）
