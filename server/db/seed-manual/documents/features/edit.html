编辑功能

PM 可以修改已有功能的标题、描述、分类归属和章节结构。

操作步骤

  1. 在功能列表中，点击功能行右侧的设置按钮
  2. 在编辑对话框中修改信息：
    • 标题和描述 — 修改功能的展示信息
    • 分类归属 — 将功能迁移到其他分类或设为未分类
    • 章节列表 — 添加新章节、删除已有章节或拖拽调整顺序
  3. 点击保存

重要提示

  • 删除章节会导致该章节的文档变为"孤立文档"。建议先将文档内容迁移后再删除章节
  • 章节标题不可直接修改 — 如需修改章节标题，需要删除旧章节并重新添加。删除后原章节的文档会变为孤立文档，需注意处理
  • 添加章节后，新章节的文档会在首次编辑时自动创建
