创建功能

PM 可以在功能列表页面手动创建自定义功能（Custom Feature）。

操作步骤

  1. 进入功能列表页面
  2. 点击自定义章节按钮
  3. 在对话框中填写：
    • 功能名称 — 必填，功能的标题
    • 所属分类 — 选填，将功能归入已有分类
    • 功能描述 — 可选，简要说明功能用途
  4. 在章节管理区域添加章节：
    • 输入章节标题（如"操作步骤"、"注意事项"）
    • 系统自动生成章节 key（英文标识符）
    • 可以添加多个章节，支持拖拽调整顺序
  5. 点击确认创建

ID 生成

自定义功能的 ID 前缀为 custom:，后面跟随 8 位随机字符（如 custom:a1b2c3d4），与系统导入的功能进行区分。
